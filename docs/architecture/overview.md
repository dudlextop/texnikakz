# Архитектурный обзор Texnika.kz

Texnika.kz — платформа для маркетплейса спецтехники и операторов. Архитектура ориентирована на модульность и готовность к масштабированию.

## Основные сервисы
- **Web (Next.js 14)** — клиентский фронтенд, SSR/ISR, i18n.
- **API (NestJS)** — BFF слой, авторизация, бизнес-логика, прокси к поиску и очередям.
- **Admin (Next.js)** — интерфейс модераторов и биллинга.
- **Workers (BullMQ)** — фоновые задачи для индексации, автобампа, уведомлений.

## Инфраструктура
- PostgreSQL 15 — основная БД (Prisma ORM).
- Redis — кеш и очереди.
- OpenSearch — поиск и фасетная выдача.
- MinIO — хранилище медиа.
- Docker Compose — локальное окружение.

Детализированная схема доменов, миграции, API-контракты и UI-флоу будут уточнены в следующих фазах разработки.
