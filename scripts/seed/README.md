# Seed Scripts Plan

В следующих этапах будут добавлены TypeScript-скрипты для:
- первичного наполнения PostgreSQL (пользователи, дилеры, специалисты, объявления, справочники)
- генерации индексов OpenSearch
- загрузки демо-медиа в MinIO

Скрипты будут размещены в этой директории и исполняться через `pnpm seed`.
